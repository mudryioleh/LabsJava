## 📋 Завдання

1. Створіть клас Person, який містить:
   a) поля fullName, age.
   б) методи move() і talk(), в яких просто вивести на консоль повідомлення -"Такий-то Person говорить".
   в) Добавте два конструктори - Person() і Person(fullName, age).
   Створіть два об’єкта цього класу. Один об’єкт ініціалізується конструктором Person(), інший - Person(fullName, age).

2. Класс Phone.
   а) Створіть клас Phone, який містить поля number, model і weight.
   б) Створіть три екземпляри цього класу.
   в) Виведіть на консоль значення їх змінних.
   г) Додати в клас Phone методи: receiveCall, має один параметр - ім'я того хто дзвонить. Виводить на консоль повідомлення “Дзвонить {ім’я}”. getNumber - повертає номер телефону. Визвати ці методи для кожного з об'єктів.
   д) Додати конструктор у клас Phone, який приймає на вході три параметри для ініціалізації полів класу - number, model і weight.
   е) Додати конструктор, який приймає на вході два параметри для ініціалізації полів класу - number, model.
   ж) Додати конструктор без параметрів.
   з) Додайте перевантажений метод receiveCall, який приймає два параметри - ім'я абонента (того, хто дзвонить) та номер телефону. Викличіть цей метод.
   к) Створити метод sendMessage з аргументами змінної довжини. Даний метод приймає на вході номера телефонів, яким буде надіслано повідомлення. Метод виводить на консоль номерів цих телефонів.

3. Реалізувати ієрархію

4. Вар. 7. Створити клас, що описує багаж пасажира. Дані класу: кількість речей і загальна вага речей. Відкриті функції класу: введення кількості речей і ваги кожної речі окремо, виведення на екран загальної ваги багажу.
