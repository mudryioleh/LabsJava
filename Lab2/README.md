# Лабораторна робота №2: Розгалуження та оператори вибору (Java)

## 📝 Опис проєкту

Ця лабораторна робота присвячена вивченню умовних операторів (`if-else`) та оператора вибору (`switch`) у Java. Проєкт демонструє вміння розв'язувати алгоритмічні задачі, працювати з математичними функціями (клас `Math`) та безпечно зчитувати дані з консолі.

# 📂 Архітектура

```
src
    ├── App.java                    # Точка входу
    └── services                    # Бізнес-логіка
        ├── InputHelper.java
        ├── LogicTaskService.java
        ├── MathTaskService.java
        └── SwitchTaskService.java
```

**services/** — класи з логікою обробки даних.

## 📋 Завдання

1. Розв'язання квадратного рівняння з обробкою різних значень дискримінанта.
2. Визначення квадранта для заданих координат $(x, y)$.
3. Логічна перевірка числа (двозначне та парне).
4. Індивідуальні завдання:
   - **4.1**: Обчислення шматково-заданої функції $f(x)$ з використанням `Math.pow`, `Math.sqrt` та `Math.log`.
   - **4.2**: Комплекс задач на оператор `switch` (календарні задачі, розклад пар, інтерактивні текстові меню калькулятора).
