# Лабораторна робота №2: Використання умовних операторів

## 📝 Опис проєкту

Отримати навики побудови алгоритмів розгалуженої структури за допомогою умовного оператора if та оператора вибору switch.

# 📂 Архітектура

```
src
    ├── App.java                    # Точка входу
    └── services                    # Бізнес-логіка
        ├── InputHelper.java
        ├── LogicTaskService.java
        ├── MathTaskService.java
        └── SwitchTaskService.java
```

**services/** — класи з логікою обробки даних.

## 📋 Завдання

1. Скласти програму для знаходження розв’язку квадратного рівняння.
2. Визначити номер квадранта, в якому знаходиться точка з заданими координатами х, у.
3. Розв’язати індивідуальні завдання згідно варіанту. Варіанти завдань
4. Індивідуальні завдання:
   - **4.1**: Використовуючи умовний оператор, скласти програму для обчислення f(x) , х ввести з клавіатури
   - **4.2**: Використовуючи оператор вибору, скласти програми для наступних задач
