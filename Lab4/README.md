# Лабораторна робота №4: Робота з масивами

## 📝 Опис

Отримати навики роботи з масивами даних.

## 🛠 Вимоги

- **Java Development Kit (JDK)**: версія 25 або вище
- **Git**: для клонування репозиторію

## 📂 Архітектура

```
src/
├── App.java              # Точка входу
└── services/             # Бізнес-логіка
    └── TaskService.java
```

**services/** — класи з логікою обробки даних.

## 📋 Завдання

1. Створити пустий масив та :
   Заповнити його 50 парними числами.
   Заповнити його 50 непарними числами.

2. Зробити масив з 10 чисел [2,17,13,6,22,31,45,66,100,-18] та:
   перебрати його циклом while
   перебрати його циклом for
   перебрати циклом while та вивести числа тільки з непарним індексом
   перебрати циклом for та вивести числа тільки з парним індексом
   вивести масив в зворотньому порядку.

3. Обчислити в одновимірному масиві, що складається з n дійсних елементів, суму елементів масиву.

4. Змінити знак всіх непарних елементів масиву.

## 🚀 Запуск

### Компіляція

```bash
cd Lab4
javac -d bin src/**/*.java
```

### Виконання

```bash
java -cp bin App
```
