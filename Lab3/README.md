# Лабораторна робота 3: Складання програм циклічної структури

## 📝 Опис проєкту

Отримати навики побудови алгоритмів циклічної структури за допомогою оператора while та for.

# 📂 Архітектура

```
src
    ├── App.java                    # Точка входу
    └── services                    # Бізнес-логіка
        └── TaskService.java
```

**services/** — класи з логікою обробки даних.

## 📋 Завдання

1.  Взяти будь-яку фразу на Ваш вибір,

- вивести її на консоль 50 разів використовуючи цикл for
- вивести її на консоль 50 разів використовуючи цикл while

2.  Використовуючи цикл for вивести на консоль інформацію:

- 0 h 0 min
- 0 h 1 min
- ...
- 2h 59 min

3. Використовуючи цикл while вивести на консоль інформацію:

- 0 h 0 min
- 0 h 1 min
- ...
- 2h 59 min

4. Використовуючи циклb while або for вивести на консоль інформацію:

- 0 h 0 min 0 sec
- 0 h 1 min 1 sec
- ...
- 2h 59 min 59 sec

5. Протабулювати функцію в заданому проміжку. Вивести кількість значень функцій, що знаходяться в діапазоні, границі якого вводяться з клавіатури ( і вивести повідомлення про їх відсутність). Програму реалізувати 2 способами: з оператором while і оператором for.
