Для запуску будь-якої лабораторної роботи з цього репозиторію вам знадобляться:

**JDK** версії 25 або вище.

## Як працювати з репозиторієм

1.  **Клонування**:
    ```bash
    git clone https://github.com/mudryioleh/LabsJava.git
    ```
2.  **Перехід до роботи**: Кожна папка (`Lab1`, `Lab2` тощо) є незалежним Java-проєктом.
3.  **Документація**: Кожна папка містить власний `README.md` з детальними інструкціями по збірці та запуску саме цієї частини коду.
