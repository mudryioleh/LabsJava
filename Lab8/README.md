# Лабораторна робота 8: Потоки вводу/виводу. Робота з файлами

## 📝 Опис проєкту

Опрацювання основних класів і методів роботи з потоками та файлами.

# 📂 Архітектура

```
src
    ├── App.java                     # Точка входу
    └── service                      # Бізнес-логіка
        ├── FileContentReader.java
        ├── FileCreator.java
        └── RandomNumberFileWriter.java
```

**services/** — класи з логікою обробки даних.

## 📋 Завдання

1. Реалізувати програму, що записує в файл ( типу txt) послідовність випадкових чисел у діапазоні, що вказаний за допомогою параметрів командного рядка. Ім’я файлу вказується також в командному рядку. У програмі передбачити перевірку існування файлу з заданим ім'ям, в разі його відсутності попередньо перед записом в файл, створити його. 


2. Прочитати інформацію з файлу, створеного у попередньому завданні. Вивести її на консоль.

## 🚀 Встановлення та запуск

### Компіляція

```bash
cd Lab8
javac -d bin src/**/*.java
```

### Виконання

```bash
java -cp bin App
```
