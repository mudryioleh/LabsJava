# Лабораторна робота №5: Побудова методів

## 📝 Опис

Отримати навики побудови методів та роботи з ними на мові Java.

## 🛠 Вимоги

- **Java Development Kit (JDK)**: версія 25 або вище
- **Git**: для клонування репозиторію

## 📂 Архітектура

```
src/
├── App.java              # Точка входу
└── services/             # Бізнес-логіка
    └── TaskService.java
```

**services/** — класи з логікою обробки даних.

## 📋 Завдання

1. Створити метод, що обчислює та повертає площу кола

2. Створити метод, що приймає три числа виводить та повертає найменше

3. Створити метод, що виводить масив

4. Створити метод, що повертає найбільше число з масиву

## 🚀 Запуск

### Компіляція

```bash
cd Lab5
javac -d bin src/**/*.java
```

### Виконання

```bash
java -cp bin App
```
